# ROM::Mongo

MongoDB support for [Ruby Object Mapper](https://github.com/rom-rb/rom)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rom-mongo'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rom-mongo
